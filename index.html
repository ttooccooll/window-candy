Help Topics
×
Welcome to Window Candy!
A polyphonic synthesizer with authentic Windows 95 styling.
Getting Started
1. Click on the keyboard or use your computer keyboard (A-L for white keys, W,E,T,Y,U for black keys)
2. Adjust the waveform, ADSR, filter, and effects to shape your sound
3. Press Z/X to change octaves
Windows
Oscillator - Main sound control with waveform selection
ADSR - Envelope shape (Attack, Decay, Sustain, Release)
Filter - Low/High/Band pass filter with cutoff and resonance
LFO - Low Frequency Oscillator for modulation
Effects - Reverb and Delay processing
Osc Mixer - Mix up to 3 oscillators
Pitch - Pitch bend, portamento, and vibrato
Scope - Real-time waveform display
Arpeggiator - Automatic note sequencing
Keyboard Shortcuts
Playing:
A-L, W,E,T,Y,U - Play notes
Z/X - Octave down/up
←/→ - Pitch bend
Space - Sustain pedal
Sound:
1-4 - Select waveform
` - Toggle arpeggiator
Other:
Escape - All notes off (Panic)
F11 - Full screen
Ctrl+S/O - Save/Load presets
Presets
Use the Start menu → Programs to browse built-in presets. Click Save Current... in the File menu to save your own sounds. Presets are stored in your browser.
MIDI Support
Connect a MIDI keyboard or controller. Use Options → MIDI Setup to detect devices. Supports note on/off, velocity, sustain pedal (CC64), and modulation wheel (CC1).

Features Guide
×
MIDI Support
Connect a MIDI keyboard or controller to play. Supports note on/off, velocity, sustain pedal (CC64), and modulation wheel (CC1).
✓ Status bar shows connected device names
✓ Velocity display shows note velocity
Touch Pressure
On touch devices, pressing harder produces louder notes. The keyboard visual shows which keys are pressed.
Sustain Pedal
Hold notes even after releasing keys. Use Spacebar or a MIDI sustain pedal (CC64).
✓ SUST indicator lights up when active
Arpeggiator
Press notes and the arpeggiator plays them in sequence. Use the Toggle button or ` key to enable.
✓ Tap Tempo: Click to set BPM by tapping
✓ Hold: Keep playing last notes even after releasing
✓ Octave Range: 1-4 octaves for wider range
✓ Patterns: Up, Down, Up/Down, Down/Up, Random, Chord
Presets
Save and load your sound settings.
✓ Save/Load to browser storage (Ctrl+S/O)
✓ Export/Import to/from JSON file
✓ Includes ALL settings: oscillators, filter, LFO, effects, arpeggiator, panning
Panning
Move the stereo position of all voices left or right. Display shows C (center), L50, R25, etc.
Modulation
MIDI modulation wheel (CC1) adds pitch vibrato to played notes. Works with the LFO for richer sound.
Visual Feedback
The keyboard visually shows which keys are pressed. Status bar shows waveform, octave, voice count, MIDI status, velocity, and sustain.
